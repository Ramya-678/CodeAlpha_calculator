# Basic Calculator

## Project Description
This is a Basic Calculator web application developed using HTML, CSS, and JavaScript.  
It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.  
The calculator has a modern UI design and is easy to use.

## Features
- Addition, Subtraction, Multiplication, Division  
- Clear (C) button to reset calculation  
- Delete/Backspace button  
- Responsive and clean UI design  
- Smooth button interactions  

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## Project Structure
calculator  
│  
├── calculator.html  

## How to Run the Project
1. Download or open the project folder  
2. Open the file named calculator.html  
3. The calculator will open in your web browser  

## Output
The webpage displays a basic calculator interface.  
Users can enter numbers and operators to perform calculations.  
The result is shown on the display screen.
